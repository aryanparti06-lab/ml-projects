# Machine Learning Model for Prediction of Prices of Commodities like Bhindi(Ladies finger), Cabbage, and Mango.

## Problem Statement
The objective of this project is to predict prices using historical data and machine learning techniques to help in better decision-making.

## Dataset
- Source: Kaggle
- Description: The dataset contains historical price information and related features.

## Approach
1. Data cleaning and preprocessing  
2. Filtering the dataset  
3. Feature engineering  
4. Model training using Random Forest Regressor  
5. Model evaluation using MAE, MSE, R2 score
